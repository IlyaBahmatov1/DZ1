Git

  • git init - Инициализирует новый репозиторий Git в текущей директории
  • git clone<ссылка> - Клонирует удаленный репозиторий Git на локальный компьютер.
  • git commit -m "<сообщение>" - Создает коммит с сохраненными в индексе изменениями и прикрепляет к нему сообщение о коммите.
  • git status - Отображает состояние вашего репозитория, включая измененные, добавленные и неотслеживаемые файлы.
  • git log - Показывает историю коммитов в репозитории.
  • git branch - Отображает список веток в репозитории, текущая ветка будет выделена.
  • git checkout <ветка> - Переключается на указанную ветку.
  • git merge <ветка> - Объединяет указанную ветку с текущей веткой.
  • git pull - Извлекает изменения из удаленного репозитория и объединяет их с текущей веткой.
  • git push - Отправляет коммиты в удаленный репозиторий.
  • git remote add <имя><ссылка> - Добавляет ссылку на удаленный репозиторий под указанным именем.
  • git remote -v - Показывает список удаленных репозиториев и их URL.
  • git fetch - Загружает изменения из удаленного репозитория без их объединения с текущей веткой.
  • git reset <файл> - Снимает указанный файл с индекса, отменяя его подготовку к коммиту.
